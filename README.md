# Image Capture and Video Processing Using OpenCV

## Overview

This experiment demonstrates real-time image capture and video processing using OpenCV in Python. The program captures frames from a webcam and performs basic video processing operations such as image saving, resizing, and rotation.

---

## Objective

The objective of this experiment is to:

- Capture images from a webcam
- Display live video streams
- Save captured frames as image files
- Resize video frames
- Rotate video frames using OpenCV

---

## Software Requirements

- Python 3.x
- OpenCV (`cv2`)
- NumPy
- Matplotlib
- Jupyter Notebook / VS Code

---

## Concepts Used

The experiment uses the following concepts:

- Webcam initialization
- Frame acquisition
- Real-time video display
- Image resizing
- Image rotation
- Image saving using OpenCV

---

## Workflow

### Step 1
Import the required Python libraries.

### Step 2
Initialize the webcam using OpenCV.

### Step 3
Capture frames continuously from the webcam.

### Step 4
Display the original video stream.

### Step 5
Resize the captured video frames.

### Step 6
Rotate the video frames.

### Step 7
Save a captured frame as a JPG image.

### Step 8
Release the webcam and close all display windows.

---

## OpenCV Functions Used

| Function | Purpose |
|---|---|
| `cv2.VideoCapture()` | Access webcam |
| `cap.read()` | Capture video frame |
| `cv2.imshow()` | Display image/video |
| `cv2.imwrite()` | Save image |
| `cv2.resize()` | Resize frame |
| `cv2.rotate()` | Rotate frame |
| `cv2.destroyAllWindows()` | Close windows |
